This writing prompt was created by Kevin Ritz 9/6/2022

  1. What are 5 reasons it is so important to learn and understand HTML?

    2. HTML is so important because:

    1. It creates structure for webpages.
    2. Every time we go on the internet HTML is present.
    3. It makes webpages viewable.
    4. It allows you to quickly generate a webpage.
    5. It allows you to have uniform formatting.

  2. In your own words, how would you describe what HTML is?

    3. HTML is an element based web format. It allows you to use tags to individually format a webpage for a more user friendly structure and break down each component and be very concise with what you would like to do.

  3. In your own words, how would you describe what a “webpage” is to someone with no knowledge of HTML?

    4. A webpage is a page with text formatted to display information. HTML allows you to take that page and build it into sections and deliver that page in a very attractive/user friendly manner.
